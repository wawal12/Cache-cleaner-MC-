## 📥 Установка

1. Скачайте файл `МС_УСТАНОВЩИК.exe` из этого репозитория
2. Запустите его и нажмите "Установить"
3. После установки на рабочем столе появится ярлык **"МС"**

## 🚀 Возможности

- Очистка кэша браузеров
